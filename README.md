# BiblioTech 📚

BiblioTech é um sistema completo de gerenciamento de biblioteca desenvolvido em Python, integrando um painel administrativo via terminal com uma interface web acessível por dispositivos móveis. O sistema utiliza FastAPI, WebSocket e MySQL para oferecer gerenciamento em tempo real de livros, usuários, empréstimos e favoritos, com autenticação segura e acesso facilitado por QR Code.
